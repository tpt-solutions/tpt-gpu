## 📦 Pre-Publication Steps

### Testing (Optional but Recommended)
1. Run full test suite: `cargo test -p tptb-core`
2. Test CLI functionality: `cargo run -p tptb-cli -- check examples/01_hello_tensor.tpts`
3. Verify documentation: `cargo doc --no-deps -p tptb-core`

### Crates.io Publication
1. Create crates.io account and get API token
2. Login: `cargo login <token>`
3. Publish in dependency order:
   ```bash
   cd layer7_tptb/tptb-core
   cargo publish
   cd ../tptb-cli
   cargo publish
   cd ../tptb-lsp
   cargo publish
   cd ../tptb-format
   cargo publish
   ```
4. Verify on https://crates.io

### GitHub Release
1. Create a new release on GitHub
2. Tag: `v1.0.0`
3. Title: `TPT Script v1.0.0 — Public Release`
4. Description: Copy from RELEASE_NOTES.md
5. Attach binaries (auto-generated by release.yml)
6. Publish release

### Post-Release
1. Update documentation website (if applicable)
2. Announce on social media / mailing lists
3. Update Docker images (if applicable)
4. Notify package managers (Homebrew, etc.)
